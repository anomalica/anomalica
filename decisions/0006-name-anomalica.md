# 0006. Name the platform Anomalica

Date: 2026-03-21
Status: accepted

## Context

The project needs a name that works across languages, signals a reference work, and does not immediately suggest "UFO enthusiast site." The working title was "NHI Exchange."

Eighteen name candidates were researched across different approaches (descriptive, conceptual, coined words, domain hacks). Domain availability was checked for .is, .org, and .com for each candidate.

## Decision

The name is **Anomalica**. It is a coined word combining "anomaly" with the "-ica" suffix used by encyclopaedic reference works (Britannica, Ars Technica, Botanica). The name covers the subject broadly (anomalous phenomena including consciousness, NHI, and reality) without narrowing to aerial phenomena.

Alternatives considered:
- NHI Exchange - the original working title. Good word but reads as financial/currency.
- Threshold - too generic, could apply to any topic.

## Consequences

The name is unique with no existing conflicts. It is pronounceable across languages. The "-ica" suffix sets expectations about the type of platform (reference work, not blog or news site).
