Which languages cover the most people? Analysis from Unicode CLDR open data.

Cumulative Coverage of Literate World Population

Two additional displayed languages are produced by mechanical conversion, not separate translation: Traditional Chinese (character conversion from Simplified Chinese) and American English (spelling and formatting conversion from British English). Together with the 28 translations, this produces 30 displayed languages.

Algorithmic Ranking

Each language is ranked by how many additional literate people it covers who are not already covered by any higher-ranked language, accounting for multilingual overlap within each territory.

Caveat: CLDR data measures functional language populations, which includes spoken fluency. For some languages (particularly pidgins and regional spoken languages), the number of people who read and write the language is significantly lower than the number who speak it. These cases are noted in the editorial adjustments below.

Methodology

Source data: Language population data from the Unicode Common Locale Data Repository (CLDR). For each of 244 territories, CLDR provides the percentage of the population that speaks each language, the territory's total population, and its literacy rate. CLDR is maintained by the Unicode Consortium and is freely available under the Unicode Licence.
Coverage model: For each territory, we calculate the literate population (population multiplied by literacy rate). When a language is added to the supported set, it covers a fraction of each territory's literate population. People who already speak a previously selected language are not counted again. Overlap between languages within a territory is modelled using an independence assumption: if 60% speak language A and 40% speak language B, the combined coverage is 1 - (1 - 0.6) x (1 - 0.4) = 76%, not 100%.
Selection algorithm: Languages are selected greedily. At each step, the language that would cover the most currently-uncovered literate people across all territories is chosen. This is a standard greedy approximation to the set cover problem, which is known to produce near-optimal results.
Limitations: The independence assumption for bilingual overlap is a simplification. In practice, speakers of minority languages within a territory are more likely to also speak the majority language than vice versa. This means our incremental estimates may slightly overcount coverage for smaller languages. However, the ranking order is robust to this effect.
Reproduce this analysis: The analysis script (analyse-languages.py) and its output (language-coverage.json) are in the repository. Run python3 analyse-languages.py with the CLDR supplementalData.xml to regenerate the results. The script, data source, and methodology are all open.
